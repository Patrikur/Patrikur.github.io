Projects

Grotenburg-Supporters

Volunteer work for a nonprofit organisation that renovates my hometowns football clubs stadium. Development, deployment and maintenance of web page and database for blog, press archive and sponsors section. Currently working on implementing headless-CMS.

eNay - The Lazy Onlineshop

Skill showcase building an online shop that does not want the visitors to shop using the Fake Store API. Inspired by the independent video game "There is no game", the shop makes it hard for you to execute an order using different hurdles and making the site unattractive to visitors. Give it a try, good luck!

Github Finder

React App using the Github API to search its users, fetching profile data and links to repositories. Built this project in React Front To Back Course by Brad Traversy and deployed on Netlify.

Family Feud Browser Game

Creation of a Family Feud browser game for regular quizzes with friends. The game is intended to support a real-life quiz as a gameboard showing questions, answers and current points per team.
